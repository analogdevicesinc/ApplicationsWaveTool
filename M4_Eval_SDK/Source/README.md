## Folder structure
```
├── bin
|     # Binary output directory
├── build
|     # Temporary build file/output
├── cmake
│     # Cmake-specific functions, scripts, etc
├── doc
│   ├── doxygen
│         # Doxygen output generated by make
├── inc
├── src
└── test
```
## Building
```bash
cd build/$platform
cmake ../..
make
```
