Building From Source

Python and Windows

Overview

The SDK build system has several configurations:

  • Native C++ static library
  • Python bindings (native dynamic library and importable .py)
  • This documentation

Pre-requisites

  • CMake
  • Compiler and build system supported by CMake on your platform
    • Linux
      • make (preferred)
      • GCC or Clang (Clang preferred)
    • Windows with Cygwin
      • make
      • GCC
    • Windows without Cygwin
      • Visual Studio 2017 (preferred)
  • SWIG (used to generate bindings for Python)
  • PythonLibs 3.7+ (to build Python bindings)
  • Doxygen (to build this documentation)

Building in windows

cd gen3_sdk/build/windows
cmake ../..
make(Windows with Cygwin) / Build gen3sdk.sln solution in Visual Studio(Windows without Cygwin)

Building in linux

cd gen3_sdk/build/$platform
cmake ../..
make

Output in windows

C++ Static Library

gen3_sdk/bin/windows/Win32/cpp/gen3_sdk.lib

Python bindings

gen3_sdk/bin/windows/Win32/python/3.7/gen3_sdk.py

gen3_sdk/bin/windows/Win32/python/3.7/_gen3_sdk.pyd

Output in linux/mac

C++ Static Library

gen3_sdk/bin/linux/x86/cpp/gen3_sdk.a

Python bindings

gen3_sdk/bin/linux/x86/python/3.7/gen3_sdk.py

gen3_sdk/bin/linux/x86/python/3.7/_gen3_sdk.so
